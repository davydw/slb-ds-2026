# Welcome to the Data Toolkit module! 👋
Over the next units, we will cover the fundamental tools that any Data Scientist or Data Analyst uses daily.

## Data Analysis
* Say “bye” to VS Code (for now), and “hello” to **Jupyter Notebook** - an IDE (Integrated Development Environment) that allows you to add code, comments, visualizations, and images inline, making it a great tool for data analysis and exploration
* Speed up your data operations thanks to **vectorized computations** and the **NumPy** Python library
* Start learning the analytical and data manipulation capabilities of **Pandas**, one of the most widely used Python data libraries


## Data Sourcing in Pandas
This is where you learn how to collect data - which can be a complex topic - from various sources, such as **CSV files**, **API**s (Application Programming Interface), and through **scraping websites**.

* Learn to utilize the **Pandas** sourcing and parsing toolkit
* Work with a wide range of data sources like **CSV**, **API**, and **web scraping**
* Extract patterns from text data using **regular expressions**

## Coding Properly & Reboot
In this unit we’ll take our coding skills to the next level:

* Object-Oriented Programming
* Testing and Test Driven Development
* Good coding style, documentation and type hints

This is also your opportunity to solidify your Python and Pandas skills in the **Reboot**.

## Statistics & Probabilities
In this unit, we cover the essential statistics that you will need as a Data Scientist, and that we’ll apply in the Decision Science module, but also later on in Machine Learning, and Deep Learning.

* Learn about descriptive statistics
* Get a big summary of statistics in general, to ensure that you have a good foundation
* Random variables and the Central Limit Theorem (CLT)

## SQL for Data Science
In this unit we dive further into **databases** and **SQL** (Structured Query Language).

* Different types of JOINs
* Window functions for advanced calculations
* Common Table Expression to break down your SQL code in smaller pieces

We’ll also discover Olist’s data structure before we use it in the Decision Science module
